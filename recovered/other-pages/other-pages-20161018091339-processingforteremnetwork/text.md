# ПО для сети терминалов — Software for Self-Service

Комплекс серверного и терминального ПО для сетей платежных терминалов.

Продаем или сдаем в аренду серверное ПО собственного производства.

Описание

Состав комплекса программ управления сетью терминалов:

ПО для терминалов или автоматических кассиров;

ПО сервера мониторинга, проведения платежей и управления терминалами;

Веб-кабинет для мониторинга и управления терминалами;

Шлюзы в различные системы, их переключение онлайн.

Возможности

Возможности управления терминалами через веб-кабинет:

Веб-мониторинг, отображение технического состояния терминалов;

Отображение онлайн остатков и балансов в платежных системах;

Отображение в веб-кабинете таблицы принятых платежей и их статусов проведения в режиме онлайн;

Удобный поиск по платежам, возможность корректировки, хранение истории попыток изменения и перепроведения транзакций;

Быстрая установка и смена сертификатов ключей на терминалы посредством одноразовых паролей;

Веб-кабинет для настройки меню терминала, редактирование списка категорий и платежей в меню на главной и других страницах терминала;

Возможность сохранения нескольких вариантов (шаблонов) меню для терминалов и моментальной загрузки любого созданного меню на любой терминал;

Управление комиссией и/или фиксированной наценкой отдельно для каждого получателя;

Хранение нескольких вариантов настройки схем комиссии с возможностью их моментальной загрузки на любой терминал;

Переключатель шлюзов в режиме онлайн;

Отчеты по инкассациям, по итогам дня или любого другого периода, в разрезе по терминалам, сводка за любой период по провайдерам.

Опционально — поддержка иерархии субагентов и ведение их балансов, веб-форма для ручного пополнения/списания по счетам субагентов;

Доступ главного агента к веб-кабинетам своих субегентов с возможностью управления их терминалами и платежами.
