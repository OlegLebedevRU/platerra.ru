# Платежный агент — Software for Self-Service

Вариант типового решения — прием всех видов платежей за сотовую связь, интернет, ЖКХ, госуслуги и т.п.

Для деятельности по приему платежей необходимо:

действующее юридическое лицо или ИП;

действующий спецсчет платежного агента, открытый в любом банке РФ;

договор с одним (или с несколькими) из платежных операторов типа Киберплат, Киви, Дельта телеком, Кредитпилот;

активированный доступ в портал платежного оператора, активированный доступ к приему платежей через АПИ и ключи для соответствующей платежной точки.
