Наши проекты

Наша компания оказывает полный спектр услуг. У нашей компании прекрасная репутация, и мы уже успели завоевать доверие и расположение своих клиентов.

Сеть терминалов банка

Один из долгосрочных проектов нашей компании. В составе этого проекта ПО для терминалов оплаты, ПО для процессинга платежей в сети терминалов, система управления терминалами. Мы осуществляем постоянное сопровождение и развитие проекта. Количество услуг и их актуальность растут.

Сеть самоинкассации через терминалы банка

Разработали ПО для терминалов инкассации и центрального сервера ситуационного центра.

Система Электронный кассир

Наша компания разрабатывает и поддерживает ПО для систем Электронного кассира. ПО включает в себя программу для сенсорного платежного терминала и личный кабинет на сервере.

Софт для сети терминалов Puntopago.net (300 точек, Панама)

Реализован проект полного комплекса программного обеспечения для сети терминалов в Панаме. Сеть на базе нашего ПО выросла с "0" до 300 терминалов и привлекла крупного инвестора. Проведены интеграции с крупнейшими биллингами региона Центральной Америки.

Проекты автоматизации салонов красоты
