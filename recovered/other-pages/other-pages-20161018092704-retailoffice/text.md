# Пункт выдачи заказов — Software for Self-Service

Задача — организовать удобную и быструю оплату заказов для ритейлера с интенсивным потоком операций оплаты.

Терминал должен определять номер документа по штрих-коду, запрашивать данные заказа по документу, принимать оплату и опционально — выдавать сдачу. Также, обеспечивать задачи кассового учета, учета инкассации, самодиагностику и онлайн-телеметрию оборудования терминала для службы технической поддержки.
