# Услуги: стратегия ИТ-развития — Системы самообслуживания

Стратегия ИТ-развития бизнеса

ИТ-стратегия организации, представленная в виде дерева целей, содержит ориентиры, касающиеся развития информационной системы. В рамках SMART-ориентиров строится сценарное прогнозирование системы. В узлах принятия решений применяются реальные опционы. Ниже приведены четыре ориентира информационно-технологической стратегии:

Возвратность инвестиций (ROI)

Надежность (SLA)

Полная стоимость владения (ТСО)

Соответствие регулятивным нормам
