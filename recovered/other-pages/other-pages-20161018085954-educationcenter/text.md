# Учебный центр — Software for Self-Service

Задача — заменить кассира в учебном центре, облегчить и повысить доступность услуг.

Для получения указанного результата использовано ПО FORPAY с настройкой меню и форм ввода для специфики указанного учебного центра.

Описанная выше задача является типовой для нашего терминального ПО FORPAY и использует базовую конфигурацию «Электронный кассир».
