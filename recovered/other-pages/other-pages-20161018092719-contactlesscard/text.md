# Бесконтактные карты — Software for Self-Service

Задача — организовать прием платежей по выставленным счетам, а также прием средств от таксистов за аренду авто таксопарка, входящего в группу компаний. В составе оборудования терминала используются не типовые устройства:

ридер бесконтактных карт MIFARE для идентификации таксистов;

сканер штрих-кода для идентификации выставленных счетов;

устройство рециклингового приема и выдачи купюр SmartPayOut;

устройство выдачи монет SmartHopper.

Для обеспечения полного цикла автоматизации мы создали модули интеграции терминального ПО с центральными серверами, содержащими базы данных по учету таксистов и их лицевых счетов, а также базы данных счетов, выставляемых для оплаты услуг автосервиса.
