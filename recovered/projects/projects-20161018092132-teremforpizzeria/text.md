# Пиццерия — Software for Self-Service

Решение для пиццерии — удобный выбор, оплата и отправка заказа в пекарню посредством смс

Задача проекта состояла в упорядочивании очереди заказов, помощи клиентам в осуществлении удобного выбора и оплаты пиццы, обеспечивающем прием наличных средств и регистрацию фактов заказа через смс-сервис.

Проект в своем составе включал:

Отображение меню;

Прием оплаты;

Отправка СМС на телефон пекаря с данными о заказе;

Формирование отчета по платежам;

Печать отчета по инкассации.
