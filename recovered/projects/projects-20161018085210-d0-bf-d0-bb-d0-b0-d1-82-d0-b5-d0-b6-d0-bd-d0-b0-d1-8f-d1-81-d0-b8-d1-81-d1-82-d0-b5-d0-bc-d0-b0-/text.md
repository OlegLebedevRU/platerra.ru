# Pay system in Panama — Software for Self-Service

Терминальная платежная система в Панаме (регион Центральной и Латинской Америки)

Требовалось принять участие на нулевой стадии бизнес-проекта и внедрить отказоустойчивые и надежные решения для  бизнеса платежных агентов в странах региона.

Проект включал в себя различные стадии и элементы:

развертывание ПО для терминалов и сервера;

локализация интерфейсов терминала и сервера;

развертывание серверной площадки;

создание безопасной сети серверов;

подготовка и инсталляция системного ПО на базе MS Windows Server и MS SQL Server;

инсталляция прикладного комплекса ПО FORPAY для центра управления сетью терминалов;

поддержка эксплуатации комплекса ПО;

подключение к различным сотовым операторам и платежным агрегаторам мирового и регионального масштаба (Telefonica, Claro, Digicel, C&W и другие).
