# Услуги: разработка ПО для сенсорных киосков — Системы самообслуживания

Разработка программ для сенсорных киосков

Сенсорные киоски позволяют экономить время обслуживания клиентов. В продвинутых проектах информационные киоски умеют даже продавать!

Особенности сенсорного интерфейса:

Нужно делать большие кнопки

Инерционные эффекты ускоряют понимание процесса

Нельзя перегружать экран элементами и разными цветами

Процесс обслуживания должен быть коротким и завершенным

И многие другие факторы

Помимо информационного обслуживания, электронных очередей и презентационных стендов киоски могут «работать» сборщиками опросов, порталами услуг и стендами для проведения тестирования.
