Прием банковских карт в терминалах

Интересные возможности для использования терминалов в бизнесе несет функция приема банковских карт. В терминалах с функцией приема банковских карт можно оплачивать товары или услуги стоимость которых относительно высока, либо цена является «не удобной» для вноса наличных в терминал, т.е. не кратна номиналам банкнот. Варианты применения функции приема банковских карт отличаются в зависимости от страны, по законам которой будет функционировать терминал, а в России и от конкретного банка, предоставляющего эквайринг для таких проектов.

Для работы с картой через систему Юнителлер на терминалах должна устанавливаться поддерживаемая версия Windows:

Windows Embedded Standard 7 x32;

Windows 8.1 x32;
