# Внедрение платежной системы в Панаме — Системы самообслуживания

Платежная система в Центральной Америке.

Зимой 2010 года нашей компанией осуществлен успешный запуск программного комплекса для зарубежного проекта по приему платежей с использованием платежных терминалов и POS-терминалов. В рамках проекта проведена интеграция с рядом крупнейших операторов сотовой связи (Cable&Wireless, Movistar, Claro, Digicel), провайдеров интернет и поставщиков других услуг. Сервис оплаты на первом этапе проекта стал доступен клиентам в 200 (двухсот) наиболее посещаемых местах, оборудованных платежными киосками самообслуживания.
