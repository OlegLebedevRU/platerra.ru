# Pay system in Guinea — Software for Self-Service

Терминальная платежная система в Экваториальной Гвинее

Задача проекта состояла во внедрении бизнеса платежных агентов в небольшой африканской стране.

Состав элементов данного проекта включал в себя:

вопросы поставки ПО для терминалов и сервера;

развертывание аппаратной платформы серверов;

создание защищенного сетевого контура;

развертывание системного ПО на базе MS Windows Server и MS SQL Server;

инсталляция прикладной платформы FORPAY с функционалом терминального процессингового центра;

организация работ по подбору и адаптации (прошивке) купюроприемников для банкнот этой страны;

подбор оборудования терминалов, организация внешнеторговой сделки и доставки оборудования.
