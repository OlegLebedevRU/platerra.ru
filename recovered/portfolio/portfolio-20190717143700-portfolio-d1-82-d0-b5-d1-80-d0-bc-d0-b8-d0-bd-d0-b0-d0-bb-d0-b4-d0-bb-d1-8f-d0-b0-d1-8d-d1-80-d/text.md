# Терминалы для аэропорта (г. Белгород) — Системы самообслуживания

Реализована оплата услуг аэропорта.

Терминал принимает наличные и банковские карты.

Терминал имеет встроенную онлайн-кассу и подключен к ОФД.

Мониторинг и отчетность по терминалу сделаны через веб-кабинет.

Ежесуточно осуществляется подготовка данных для загрузки отчета в 1С.
