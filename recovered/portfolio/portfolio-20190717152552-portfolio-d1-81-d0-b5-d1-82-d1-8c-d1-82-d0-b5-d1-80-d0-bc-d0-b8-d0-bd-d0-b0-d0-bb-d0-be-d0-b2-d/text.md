# Сеть терминалов самоинкассации — Системы самообслуживания

Решение для крупнейшего оператора инкассации НКО ФИНЧЕР.

Решение для НКО ФИНЧЕР (г.Москва и МО). У кредитной организации появилась задача упростить процесс инкассации для своих клиентов, распределенных по разным районам города. Решение состояло в том, чтобы установить на территории клиентов специальные терминалы для внесения наличных в количестве 30 штук, чтобы клиенты могли инкассировать кассу в них. А инкассаторы в плановом режиме, без необходимости согласовывать выделенное время с клиентом производят выемку наличных уже из терминалов. Для решения создан выделенный сервер процессинга для сбора и обработки транзакций внесения наличных, а также для сбора транзакций выемки кассет. Для контроля активности сети терминалов также внедрена система технического мониторинга состояния связи и оборудования терминалов.

Состав решения:

Софт для терминалов с функцией приема купюр, интерфейсом авторизации клиентов и контроля доступа инкассаторов.

Процессинговая система для приема и обработки данных по операциям самоинкассации и контролю изъятия кассет.

Ситуационный портал для мониторинга технического состояния сети.
