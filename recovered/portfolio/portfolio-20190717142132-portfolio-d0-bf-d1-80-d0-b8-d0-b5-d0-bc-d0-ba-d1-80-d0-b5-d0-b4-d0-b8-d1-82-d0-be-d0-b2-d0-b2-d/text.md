# Автоматизация банка — Системы самообслуживания

Проект по организации сети приема платежей банка.

Начат в 2010 году.

Всего развернуто более 20 терминалов оплаты.

Подключены десятки местных поставщиков услуг.

Организована оплата через сайт банка, платежные терминалы и кассы.

Решение для Банка (г.Рязань) — организация сбора наличных платежей для клиентов юрлиц. У банка возникла задача внедрить для своих клиентов ВУЗов сервис по сбору наличных платежей за услуги обучения. Для решения задачи решено использовать сеть терминалов самообслуживания и интегрировать процессинг платежей с банковской АБС. В результате решения на сеть терминалов банка установлено программное обеспечение FORPAY Terminal, со специальном интерфейсом, позволяющим принимать и обрабатывать оплату за обучение в различные ВУЗы города. Транзакции оплаты собираются в едином процессинговом центре FORPAY.Net и посредством сервиса интеграции передаются выгрузкой в АБС банка.
